# Help me, Kiddo!
 
Aplicación para Android TV que permite instalar y gestionar aplicaciones de forma sencilla, pensada para personas con poca experiencia técnica.
 
## ¿Para qué sirve?
 
Instalar apps en Android TV puede ser complicado para alguien que no está familiarizado con el proceso. Esta aplicación centraliza todo en una interfaz simple: selecciona la app, descarga e instala. Sin tecnicismos.
 
## ¿Para quién es?
 
Para cualquiera que quiera preparar un Android TV Box para un familiar o amigo sin tener que explicarle cómo funciona la instalación manual de APKs.
 
## Requisitos
 
- Android TV Box con Android 9 o superior
- Aplicación [Downloader](https://play.google.com/store/apps/details?id=com.esaban.downloader) instalada desde Play Store (necesaria para la primera instalación)
## Instalación
 
1. Instala Downloader desde Play Store en el TV Box
2. Abre Downloader e introduce la URL de la última versión disponible en [Releases](../../releases)
3. Instala y abre la aplicación
